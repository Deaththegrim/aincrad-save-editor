  • Visual pickers — face, hair, eyes, and extras chosen from the game's own thumbnails, not by number.
  • Death game (permadeath) toggle — turn the character-creation permadeath flag off (rescue a death-game character) or on, per character.
  • All 12 voices — male and female rows, usable on any body.
  • Looks — save an appearance as a named preset and re-apply it; every applied value is validated against the game's real data tables.
  • Safe by construction — working-copy edits, timestamped backups, and gates that refuse values the game would break on.
  • Built-in key recovery — reads your own pak AES key from your running game; nothing is shipped or uploaded.
  • UI in 10 languages.
